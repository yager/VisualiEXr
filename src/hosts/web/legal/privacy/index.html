2. 取得する情報

本サービスは、必要最小限の情報のみを取り扱います。

  • 拡張機能・アプリの設定：選択したビジュアライザ等の設定は、お使いの端末内（ブラウザの storage / localStorage）にのみ保存されます。当方のサーバーへ送信されることはありません。
  • 音声・映像：ビジュアライザの描画のためにブラウザ内で音声を解析しますが、録音・保存・外部送信は一切行いません。処理はすべて端末内で完結します。
  • 寄付（決済）に関する情報：寄付をいただく際の決済は、決済事業者 Stripe（Stripe Payments Japan 株式会社ほか）が処理します。当方はクレジットカード番号等の決済情報を保持しません。取引の確認のため、Stripe を通じて金額・日時・メールアドレス等の取引情報を受領する場合があります。
  • お問い合わせ情報：メール等でお問い合わせをいただいた場合、返信のために氏名・メールアドレス・内容を取得します。
  • アクセスログ：本ウェブサイトは GitHub Pages 上で提供されており、閲覧に伴う通信記録（IP アドレス等）はインフラ提供者側で記録される場合があります。

3. 利用目的

  • 本サービスの提供・維持・改善のため
  • お問い合わせへの対応のため
  • 寄付に関する取引の確認・記録および会計・税務対応のため
  • 不正利用の防止のため

4. 第三者への提供・委託

当方は、法令に基づく場合を除き、取得した情報を本人の同意なく第三者に提供しません。ただし、本サービスの提供に必要な範囲で、以下の外部サービスを利用（業務委託）しています。各社における情報の取り扱いは、各社のプライバシーポリシーに従います。

  • Stripe（決済処理）
  • GitHub / GitHub Pages（ソースコード公開・ウェブサイトの配信）
  • Google Chrome ウェブストア（拡張機能の配布）

5. Cookie・アクセス解析

本ウェブサイトは、独自のアクセス解析ツールや広告目的の Cookie を使用していません。決済ページ（Stripe）等の外部サービスでは、それぞれのサービスが Cookie 等を使用する場合があります。

6. 情報の管理・安全対策

当方は、取り扱う情報について、漏えい・滅失・毀損の防止に努めます。決済情報は PCI DSS に準拠した決済事業者（Stripe）が管理し、当方の端末・サーバーには保持しません。本ウェブサイトおよび決済は暗号化された通信（HTTPS）で提供されます。

7. 開示・訂正・削除等の請求

当方が保有する自己の情報について、開示・訂正・利用停止・削除等をご希望の場合は、visualiexr@gmail.com までご連絡ください。ご本人であることを確認のうえ、法令に従い対応します。

8. 本ポリシーの変更

当方は、必要に応じて本ポリシーを変更することがあります。変更後の内容は本ページに掲載した時点から効力を生じます。

9. お問い合わせ

本ポリシーに関するお問い合わせは visualiexr@gmail.com までお願いいたします。
