VisualiEXr

音にあわせて動くグラフィックをブラウザだけで。インストール不要、URLを開けばすぐ試せます。

Audio-reactive visuals, right in your browser. No install — just open the URL and try it.

内蔵ビジュアライザBuilt-in Visualizers

    注記Notes

    • このライブデモは マイク／タブ音声 に反応します。YouTube 動画への直接重ね表示は Chrome 拡張版の機能で、Web版にはありません。
      This demo reacts to mic / tab audio. Overlaying directly on YouTube videos is a Chrome-extension-only feature, not available here.
    • タブ音声の取得（getDisplayMedia）は主に Chrome 系ブラウザ向けです。非対応ブラウザではマイクのみお使いください。
      Tab-audio capture is Chrome-centric; unsupported browsers can still use the microphone.
    • 音声は解析のみに使われ、サーバーへ送信・保存されません（すべてブラウザ内で完結）。
      Audio is analyzed locally only — nothing is uploaded or stored.
